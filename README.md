# OpenData

Some datasets I needed for my projects.

## Compiled files

The compiled files can be found in [data/](./data/).

## Country Codes

- [XRepository](https://www.xrepository.de/details/urn:xoev-de:kosit:codeliste:country-codes_8) hosts a german 
  country code list which is enriched with english aliases and converted into one JSON file.
- English country names from [Wikipedia](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2)
- French country names from [iso.org](https://www.iso.org/obp/ui/#search/code/)

## Language Codes

- English language names from [Wikipedia](https://en.wikipedia.org/wiki/List_of_ISO_639_language_codes)
- German language names from [Wikipedia](https://de.wikipedia.org/wiki/Liste_der_ISO-639-Sprachcodes)
- French language names from [Wikipedia](https://fr.wikipedia.org/wiki/Liste_des_codes_ISO_639-1)

## Useful tools

- [wikitable2json](https://www.wikitable2json.com/) allows it to download tables from wikipedia posts as JSON

## Rohdaten sind geil! [^1]

Raw data are stored in [rawdata/](./rawdata/) on purpose, so that changing the enriched data files is as
easy as possible, regardless if the original sources still exist or not.
